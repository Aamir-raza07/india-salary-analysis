# India Salary Data Analysis 📊

## About
Statistical analysis of 22,770 India job records
to find salary trends across cities and job roles.

## Key Findings
- 🏆 Mumbai pays the most — ₹7,52,751 avg
- 💾 Database roles pay the most — ₹7,50,000+ avg
- 🌐 Web development pays the least — ₹3,95,000 avg
- ⭐ Company rating does NOT affect salary (correlation = 0.028)
- 🧹 5% data were outliers — removed using IQR method
- 📈 Skewness improved from 46.79 → 0.99 after cleaning

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Dataset
- Records: 22,770 rows
- Columns: 8
- Source: India Jobs & Salary Data
